# P0-P3 上线化计划

## 定位

本项目可以升级为真实可用的基金分析与决策辅助系统，但不能表述为自动化投顾或个性化投资建议系统。系统输出用于研究、筛选和比较，不构成买入、持有或卖出指令。

## P0：合规边界与数据质量

- 明确历史业绩不代表未来表现。
- 报告中加入非个性化投资建议声明。
- 为基金加入数据质量标签，例如样本期是否充足。
- 对最大回撤过深的基金加入风险预警。
- 保留所有输入数据、输出报告和运行参数，方便复现。

## P1：可用的基金分析系统

- 支持用户输入基金代码并抓取真实净值数据。
- 自动抓取基金名称，生成可读报告。
- 计算收益、波动、回撤、Sharpe、Calmar 和滚动正收益比例。
- 生成多因子评分、三类投资者画像排名和权重敏感性分析。
- 输出风险等级和决策辅助标签：重点观察、可观察、高回撤预警、暂不优先。

## P2：产品化应用

- 增加 Web 页面或 API 服务。
- 增加数据库，保存基金净值、基金元数据、指标和报告。
- 增加定时任务，每日或每周自动更新净值和报告。
- 增加日志、异常告警、接口失败重试和数据源备份。
- 支持基金类型、成立年限、规模、费用等筛选条件。

## P3：研究增强

- 加入基金持仓、行业暴露、基金经理、基金规模和费率。
- 加入基金季报、公告和经理观点文本。
- 用 TF-IDF、情感词典或 BERT 构建文本因子。
- 用 Random Forest、XGBoost 等模型预测未来区间风险收益表现。
- 用 SHAP 解释模型判断，形成可解释金融科技项目。
